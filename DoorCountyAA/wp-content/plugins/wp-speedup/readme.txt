== Installation ==

How to install the plugin and get it working:


Method-A:

1. Go to your wordpress admin "yoursite.com/wp-admin"

2. Login and then access "yoursite.com/wp-admin/plugin-install.php?tab=upload

3. Upload and activate this plugin

4. Now go to admin menu -> settings -> WP SpeedUp

	
Method-B:

1.	Download the WP SpeedUp installation package and extract the files on your computer. 
2.	Create a new directory named `wp-speedup` in the `wp-content/plugins` directory of your WordPress installation. Use an FTP or SFTP client to upload the contents of your WP SpeedUp archive to the new directory	that you just created on your web host.
3.	Login to the WordPress Dashboard and activate the "WP SpeedUp" plugin.
4.	Once the plugin is activated, a new **WP SpeedUp** sub-menu will appear in your Wordpress admin -> settings menu.

[WP SpeedUp]: http://androidbubble.com/blog/wordpress/plugins/wp-speedup
